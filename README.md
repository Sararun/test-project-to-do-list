# Установка

## Требования

- Docker
- Docker Compose

## Шаги установки

### 1. Настройка hosts файла

Добавьте в `/etc/hosts` следующую строку:

```
127.0.0.1 test-project.local
```

### 2. Конфигурация окружения

Скопируйте файл с примером переменных окружения:

```bash
cp .env.example .env
```

### 3. Запуск Docker контейнеров

Соберите и запустите контейнеры:

```bash
docker-compose up -d --build
```

### 4. Выполнение миграций

Войдите в контейнер nginx-unit:

```bash
docker exec -it <container_name> bash
```

Выполните миграции с наполнением базы данных:

```bash
php artisan migrate:fresh --seed
```

### 5. Готово!

Приложение доступно по адресу: [http://test-project.local](http://test-project.local)
